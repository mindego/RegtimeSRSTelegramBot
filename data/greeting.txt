Добрый день!

Это аварийный шлюз RegtimeSRS регистратора Webnames.ru, предназначенный для предоставления минимальных возможностей по управлению доменами для реселлеров, которые не могут управлять доменами через API или веб-интерфейс регистратора в связи с блокировками доступа.

Шлюз поддерживает следующие функции: 

Действия с доменами:
/renew domainname.tld - продление домена
/update_ns domainname.tld ns1.host.tld ns2.host.tld - смена DNS серверов домена

Получение информации
/balance - баланс аккаунта
/info domainname.tld - информация о домене

Авторизация
/login login password - данные доступа в личный кабинет/API реселлера. Если Вы опасаетесь предоставлять их через telegram - свяжитесь со службой техподдержки webnames.ru для альтернативных способов подключения авторизации.
/logout - сброс текущих установленных данных доступа в личный кабинет/API
/status - информация о текущем соединении
